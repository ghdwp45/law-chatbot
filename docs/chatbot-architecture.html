2️⃣ 데이터 구조 (Turso DB)

청크(chunk)=긴 문서를 검색용으로 잘게 자른 조각. 각 조각은 ‘의미 벡터’와 ‘키워드 색인’ 두 가지로 검색됨.

각 데이터는 두 벌로 색인됩니다 — *_chunks(의미 벡터, 비슷한 뜻 검색) + *_fts(키워드 trigram, 글자 일치 검색). 둘을 같은 rowid로 묶어두고, 검색 때 합쳐서 정확도를 올립니다.
